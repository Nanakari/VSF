VTuber Song Finder Portable

使用方法：
1. 双击 VTuberSongFinder.exe。
2. 程序会在本机启动 Web 界面，并自动打开浏览器。
3. 如果浏览器没有自动打开，请手动访问：
   http://127.0.0.1:5000
4. 数据库文件 vtuber_songs.sqlite3 必须和 exe 放在同一个目录。
5. 程序以无命令行窗口模式启动；关闭浏览器页面后，程序会在确认没有其它页面连接后自动退出。

搜索：
- 首页会列出所有频道。
- 可以按频道、歌曲、艺人 / 作者三个条件组合搜索。
- 三个条件都不填时返回首页。
- 只命中一首歌时会自动展开，多首歌时默认折叠。
- 不指定频道、结果涉及多个频道时，歌曲下面会先列出频道二级列表，频道默认折叠。
- 详情表不显示单独的时间列；点击“打开”会直接跳到对应时间点。

日志：
- 程序优先在 exe 同级目录写入 logs/app.log。
- 如果 exe 同级目录不可写，会回退到用户目录下的应用日志目录。
- 如果页面打不开、端口被占用、数据库缺失或程序启动失败，请先查看日志。
- 日志会自动滚动，最多保留 3 个备份。

说明：
- 这是本地程序，不需要安装 Python。
- 前端只读取本地 SQLite 数据库，不会自动联网抓取 YouTube。
- 如果之后要重新索引频道，建议使用源码目录里的命令行工具。
